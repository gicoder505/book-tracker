# Трекер прочитанных книг

Консольное приложение на Python для учёта прочитанных книг с хранением данных в JSON-файле.

## Функциональность

- Добавление книги (автор, название, оценка 1–5, дата прочтения)
- Проверка дубликатов при добавлении (по автору и названию)
- Просмотр полного списка книг
- Расчёт средней оценки по всем книгам
- Статистика: количество книг по каждому автору
- Удаление книги по номеру из списка

## Как запустить

```bash
python main.py
```

Требования: Python 3.7+. Сторонние библиотеки не нужны.

## Пример работы

```
=== Трекер прочитанных книг ===
1. Добавить книгу
2. Показать все книги
3. Показать среднюю оценку
4. Статистика по авторам
5. Удалить книгу
6. Выход
Выберите пункт меню: 1
Автор: Лев Толстой
Название: Война и мир
Оценка (1-5): 5
Дата прочтения (дд.мм.гггг): 15.05.2025
Книга 'Война и мир' добавлена.
```
